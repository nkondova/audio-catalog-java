# Audio Catalog (Java)

Конзолно Java приложение за управление на личен аудио каталог.

## Описание
Приложението позволява съхранение и управление на различни аудио обекти:
- Песни
- Podcast-и
- Аудиокниги
- Албуми
- Playlist-и

Всеки аудио обект има заглавие, жанр, продължителност, категория, автор и година на публикуване.

## Функционалности
- Добавяне и изтриване на аудио обекти
- Търсене по заглавие и автор
- Филтрация по жанр и година
- Сортиране по заглавие
- Създаване и управление на playlist-и
- Записване и четене на каталога от текстов файл

## Стартиране
1. Проектът се отваря в IntelliJ IDEA
2. Стартира се класът `Main`
